# Labour Hire Quote Tool

A web application for generating labour hire quotations, with multi-brand
document output and SharePoint integration.

> **Note:** This is a desensitised public copy. Company names, brands
> (e.g. "Acme Group", "Northwind Workforce"), tenant details and credentials
> are illustrative placeholders. Real values are supplied via environment
> variables (see `.env.example`) and are never committed.

## Features

* Job and project selection
* Labour cost calculations
* Shift and roster planning
* Client pricing and markup calculations
* Excel quote generation
* PDF quote generation
* SharePoint integration
* Existing quote management

## Technology Stack

* Python
* Flask
* SharePoint
* Microsoft Graph API
* Excel Automation

## Purpose

This application streamlines the labour hire quotation process by reducing manual calculations and improving consistency across project quotations.

Developed for internal business use.
